<task type="auto">
  <name>Task 2: Add client-side pagination to search results</name>
  <files>src/main/resources/templates/landing.html</files>
  <action>
**1. Raise fetch limit:** Change the search fetch URL from `'&limit=10'` to `'&limit=200'`. This gives pagination enough items to work with. The search endpoint accepts any limit value (verified in RestaurantController.java).

**2. Add pagination container:** After `<div id="search-results"></div>`, add:
```html
<div id="pagination-controls" style="margin-top:8px"></div>
```

**3. Add pagination state variables** at the top of the `<script>` block (alongside existing `lastResults`):
```javascript
var PAGE_SIZE = 20;
var currentPage = 0;
```

**4. Rewrite `renderResults` to become `renderPage`:**
- Rename the existing `renderResults(arr)` to an internal `renderResultRows(arr)` that renders the HTML rows
- Create `renderPage()`:
  ```javascript
  function renderPage() {
    var start = currentPage * PAGE_SIZE;
    var sorted = sortResults(lastResults, document.getElementById('sort-control').value);
    var slice = sorted.slice(start, start + PAGE_SIZE);
    renderResultRows(slice);
    renderPagination();
  }
  ```
- Create `renderPagination()`:
  ```javascript
  function renderPagination() {
    var totalPages = Math.ceil(lastResults.length / PAGE_SIZE);
    var el = document.getElementById('pagination-controls');
    if (totalPages <= 1) { el.innerHTML = ''; return; }
    el.innerHTML =
      '<div style="display:flex;align-items:center;gap:12px;padding:12px 0;justify-content:center">'
      + '<button onclick="goPage(-1)" style="padding:6px 14px;border:1px solid #ddd;border-radius:6px;cursor:pointer;background:white"'
      + (currentPage === 0 ? ' disabled style="padding:6px 14px;border:1px solid #ddd;border-radius:6px;background:white;opacity:0.4;cursor:default"' : '')
      + '>&larr; Prev</button>'
      + '<span style="font-size:0.88em;color:#555">Page ' + (currentPage + 1) + ' of ' + totalPages + '</span>'
      + '<button onclick="goPage(1)" style="padding:6px 14px;border:1px solid #ddd;border-radius:6px;cursor:pointer;background:white"'
      + (currentPage >= totalPages - 1 ? ' disabled style="padding:6px 14px;border:1px solid #ddd;border-radius:6px;background:white;opacity:0.4;cursor:default"' : '')
      + '>Next &rarr;</button>'
      + '</div>';
  }
  ```
- Create `goPage(delta)`:
  ```javascript
  function goPage(delta) {
    var maxPage = Math.ceil(lastResults.length / PAGE_SIZE) - 1;
    currentPage = Math.max(0, Math.min(maxPage, currentPage + delta));
    renderPage();
  }
  ```

**5. Update `doSearch()` success handler:**
Replace `renderResults(lastResults)` with:
```javascript
currentPage = 0;
renderPage();
```

**6. Update sort control change handler:**
Replace `renderResults(sortResults(lastResults, this.value))` with:
```javascript
currentPage = 0;
renderPage();
```

**7. Update `hideResults()`:** Also clear pagination:
```javascript
document.getElementById('pagination-controls').innerHTML = '';
```

**CRITICAL pitfall:** Always reset `currentPage = 0` on new search AND on sort change. Otherwise the user might be on page 3, sort changes, and fewer items exist on page 3 of the new sort order.
